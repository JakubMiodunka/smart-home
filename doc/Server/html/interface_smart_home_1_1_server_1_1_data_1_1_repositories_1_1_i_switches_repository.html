Creates a new representation of electrical switch within the repository.

Parameters
stationIdThe unique identifier of the station that controls the switch.
localIdThe identifier of the switch, unique only at the station level.
expectedStateExpected state of the switch. true if the circuit shall be closed and current shall flow; false otherwise.
actualStateActual state of the switch. true if the circuit is closed and current is flowing; false otherwise. null indicates the state is unknown.

Switch entity saved within the repository.
