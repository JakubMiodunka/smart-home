Entity representing the general details of a station functioning within the system. Used for data exchange between the server and the database.

Although ushort would perfectly fit the network port range, int is used here for consistency with the .NET ecosystem (ex. IPEndPoint.Port) and to ensure compatibility with SQL Server, which lacks a native unsigned 16-bit integer type. This avoids unnecessary type casting and simplifies data mapping.

Parameters
IdThe unique global identifier for the station.
MacAddressThe unique physical MAC address of the station, used as a primary identifier in the network.
IpAddressThe IP address assigned to the station within the network. Set to null if station is offline and address is unknown.
ApiPortThe network port on which the station's control service is listening. Shall be in range from IPEndPoint.MinPort to IPEndPoint.MaxPort. As this record serves purely as a data container, range validation is enforced at the database level via constraints. Set to null if station is offline and port is unknown.
ApiVersionVersion of the API exposed by the station. Set to null if station is offline and its API version is unknown.
LastHeartbeatTimestamp of the last heartbeat signal received from the station.
Exceptions
ArgumentOutOfRangeExceptionThrown when the value of at least one argument is outside its valid range.
