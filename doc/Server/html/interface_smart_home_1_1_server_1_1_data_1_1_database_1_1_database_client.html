Detailed Description

Database client handling interactions with the database.

This interface is primarily created to support repository classes whose capabilities may exceed pure SQL execution. Such classes shall interact with the database through this interface.

This class is dedicated solely to executing SQL code. If there is a need to introduce additional logic exceeding pure data access, it must be implemented in a separate repository class dedicated to a specific entity type.
