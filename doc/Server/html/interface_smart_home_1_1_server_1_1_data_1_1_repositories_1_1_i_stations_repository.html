Creates a new representation of station within the repository.

Parameters
macAddressMAC address of the station. Shall be unique within the repository.
ipAddressIP address of the station. Set to null if station is offline and address is unknown.
apiPortThe network port on which the station's control service is listening. Shall be in range from IPEndPoint.MinPort to IPEndPoint.MaxPort. Set to null if station is offline and port is unknown.
apiVersionVersion of the API exposed by the station. Set to null if station is offline and its API version is unknown.
lastHeartbeatTimestamp of the last heartbeat signal received from the station.
Returns
Station model saved within the repository.

Updates properties of specified station.

Currently updating station IP address is needed, but method is already prepared to support multiple properties update if it will be needed in the future.

Parameters
idSpecifies which station shall be updated.
updateIpAddresstrue if IP address of specified station shall be updated, false otherwise.
ipAddressNew IP address of the station. Ignored if value of updateIpAddress is set to false.
updateApiPorttrue if API port of specified station shall be updated, false otherwise.
apiPortNew station API port. Shall be in range from IPEndPoint.MinPort to IPEndPoint.MaxPort. Set to null if station is offline and port is unknown. Ignored if value of updateApiPort is set to false.
updateApiVersiontrue if API version of specified station shall be updated, false otherwise.
apiVersionNew station API version. Set to null if station is offline and its API version is unknown. Ignored if value of updateApiVersion is set to false.
updateLastHeartbeattrue if timestamp of last heartbeat signal shall be updated, false otherwise.
lastHeartbeatUpdated value of last heartbeat signal timestamp. Ignored if value of updateLastHeartbeat is set to false.
Returns
The updated station model saved within the repository. If the specified station does not exist, null reference is returned.
